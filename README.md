## About Me

I build and break AWS infrastructure until it's production-grade — then document it so someone else could rebuild it in five minutes instead of five hours.

My work sits at the intersection of Infrastructure as Code, CI/CD automation, and cloud security: Terraform for provisioning, GitHub Actions and Jenkins for delivery, Docker and Kubernetes for runtime, and a habit of shifting security checks as early into the pipeline as possible.

Before this, I studied AI & Data Science. That background still shows up — mostly as a data-driven lens on architecture decisions, and in occasionally deploying ML models the same way I deploy everything else: containerized, monitored, and reproducible.

**Currently open to:** Junior DevOps Engineer / Cloud Engineer roles — Pakistan, Germany, or remote.
